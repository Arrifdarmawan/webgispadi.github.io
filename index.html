Peta berbasis WEB ini disederhanakan dari Bootleaf Template bmcbride dan SinauGIS. Dengan memanfaatkan perpustakaan Bootstrap 3, Leaflet dan typeahead.js. Sehingga aplikasi ini tidak berbayar sesuai dengan prinsip Open source dan MIT licensed.

Sekilas Tentang GISTANI
  • GISTANI merupakan program yang dibangun berdasarkan penelitian dengan judul ESTIMASI PRODUKSI PADI BERDASARKAN FASE TUMBUH DAN MODEL ARIMA (AUTOREGRESSIVE INTEGRATED MOVING AVERAGE) DENGAN MENGGUNAKAN CITRA LANDSAT 8 DI KABUPATEN SUKOHARJO MENGGUNAKAN VISUALISASI WEBGIS sebagai salah satu syarat mendapatkan gelar Strata-1. Estimasi Produksi Berdasarkan Sebaran Fase Tumbuh Tanaman Padi merupakan Informasi Utama dalam WEBGIS ini. Terlepas dari tujuan praktis, harapanya di era-otomasi (Industri 4.0) WEBGIS ini dapat bermanfaat sebagai media kajian dalam menghadapi isu ketahan pangan baik regional maupun nasional bagi para Akademisi dan Praktisi dilingkungan bidang kajian Tanaman Pangan, serta dapat memperkaya pustaka ilmiah khususnya dilingkungan Fakultas Geografi UMS dan umumnya seluruh unsur, instansi maupun lembaga terkait ketahanan pangan. Adapun Model dan Metode yang digunakan dalam Proses Penelitian, Sebagai Berikut:
  • Model Indeks Vegetasi : menggunakan Model Algortima NDVI dan MSAVI
  • Analisi Regresi Linier Sederhana : Untuk Menentukan Model Algorimta Terbaik
  • Model Estimasi ARIMA : Untuk Menentukan Nilai Estimasi Produktivitas
